# Snake Game

A modern browser-based Snake Game built with **HTML**, **CSS**, and **JavaScript**. Eat food, grow longer, beat your high score, and survive as long as possible.

## Features

* Classic Snake gameplay
* Random food generation
* Persistent high score using Local Storage
* Live game timer
* Keyboard controls with visual key highlights
* Collision detection (walls & self)
* Restart game functionality
* Clean and responsive UI

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

## Getting Started

Clone the repository and open `index.html` in your browser.

```bash
git clone <repository-url>
cd snake-game
```

No additional setup or dependencies are required.

## Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |
